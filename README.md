# OpenAI Proxy

Super simple implementation of OpenAI API proxy forwarding based on FastAPI and httpx.

## Get start

### 1. Install dependencies

```shell
pip install -r requirments.txt
```

### 2. Run server

```shell
python -m uvicorn app:app
```
